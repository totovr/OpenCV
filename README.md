# OpenCV3

OpenCV3 examples using Python version 3.3.6 in Conda and OS X Sierra

# Installation

路Install Homebrew from 

    https://brew.sh/index_es.html

路Install Python 3

    https://www.python.org/

路Install Anaconda from the offical page (for Python 3.6)

    https://www.anaconda.com/download/#macos

路Install OpenCV with the next command:

    conda install -c conda-forge opencv
